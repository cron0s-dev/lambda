# λ Lambda
> A cross-platform recursive-descent math interpreter written in C.
## 📸 Preview



## 📦 Build

```bash
git clone https://github.com/cron0s-dev/lambda.git
cmake -B build -G Ninja
cd build
ninja
```

or alternatively if you don't have Ninja

```bash
git clone https://github.com/cron0s-dev/lambda.git
cmake -B build
cd build
make
```
