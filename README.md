# λ Lambda
> A cross-platform recursive-descent math interpreter written in C.
## 📸 Preview



## 📦 Build

```bash
git clone https://github.com/cron0s-dev/lambda.git
cd ./lambda
cmake -B build -G Ninja -DCMAKE_C_COMPILER=gcc
cd build
ninja
```

or alternatively if you don't have Ninja installed

```bash
git clone https://github.com/cron0s-dev/lambda.git
cd ./lambda
cmake -B build -DCMAKE_C_COMPILER=gcc
cd build
make
```
